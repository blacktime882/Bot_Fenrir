# 🔧 MAIN_FILE: Пошаговая инструкция установки

## ⚠️ Твоя проблема

```
Error: Cannot find module './index.js'
```

**Причина:** Переменная `MAIN_FILE` не установлена в HeavenCloud Settings!

---

## ✅ КАК ИСПРАВИТЬ

### Способ 1: Через Server Settings (ТОЧНО РАБОТАЕТ)

#### Шаг 1: Перейди в Server Settings

1. Откройте https://control.heavencloud.in
2. Выберите свой сервер
3. Найдите меню слева, ищите **"Settings"** или **"Server Settings"** или иконку ⚙️
4. Нажмите на неё

#### Шаг 2: Найди переменные окружения

Ищи один из этих разделов (в зависимости от интерфейса HeavenCloud):

- **"Startup"** 
- **"Environment Variables"**
- **"Startup Configuration"**
- **"Docker"**

#### Шаг 3: Найди поле MAIN_FILE

В этом разделе должны быть поля для переменных. Ищи:

```
MAIN_FILE  [текстовое поле]
NODE_ARGS  [текстовое поле]
```

или просто:

```
Переменные:
- MAIN_FILE: _____
- NODE_ARGS: _____
```

#### Шаг 4: Установи значение

В поле **MAIN_FILE** вставь:

```
index.js
```

**Просто "index.js", ничего больше!**

#### Шаг 5: Сохрани

Нажми кнопку:
- **"Save"**
- **"Apply"**
- **"Update"**
- или любую кнопку сохранения

#### Шаг 6: Перезагрузи сервер

1. Откройте консоль HeavenCloud
2. Нажми кнопку **STOP** (если сервер работает)
3. Подожди пока сервер остановится
4. Нажми кнопку **START**
5. Смотри логи - должно быть:

```
[Bot] Starting Fenrir Bot v2.0
[Bot] Logged in as BotName#0000
[Data] Loaded: 43 tiers, 44056 slots
```

✅ **Готово!**

---

### Способ 2: Если ты не нашёл Settings

Возможно, интерфейс другой. Попробуй:

1. Поищи кнопку **"Server"** в левом меню
2. Затем **"Configuration"** или **"Config"**
3. Или нажми на **название сервера** вверху
4. Затем **"Settings"**

---

## 🔍 Если всё ещё не работает

### Способ 3: Установи переменную при загрузке файлов

Если ты загружал файлы через File Manager, делай так:

1. В **File Manager** создай файл: `.env.production`

2. Содержимое:
```
MAIN_FILE=index.js
NODE_ARGS=
AUTO_UPDATE=0
```

3. Перезагрузи сервер

Может быть HeavenCloud прочитает эту переменную.

---

## 🚨 Последний вариант

Если НИЧЕГО не помогает:

1. **Свяжись с поддержкой HeavenCloud** через чат на https://control.heavencloud.in

2. **Напиши им:**
   ```
   Привет, мне нужно установить переменную окружения MAIN_FILE=index.js
   Я не вижу поля для её установки в Server Settings.
   Моё приложение Node.js не может запуститься без этой переменной.
   Можешь помочь?
   ```

3. Они помогут установить переменную вручную

---

## ✔️ Чек-лист

Перед каждой попыткой:

- ✅ `MAIN_FILE=index.js` установлена в Server Settings
- ✅ Ты нажал **STOP** → **START** (полная перезагрузка)
- ✅ Файл `index.js` существует в `/home/container/`
- ✅ Файл `.env` создан с `DISCORD_TOKEN`
- ✅ Файл `package.json` существует в `/home/container/`

Если всё ✅ - сервер должен запуститься!
