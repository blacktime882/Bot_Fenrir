# 📊 Источники данных Bot_Fenrir

## Все данные загружаются с browse.wf

Бот **НЕ содержит** никаких хардкодированных данных об арбитражах. Вся информация загружается в реальном времени с внешних источников.

## 🔄 Источники данных

### 1. **Расписание арбитражей** (Критично)
- **URL:** https://browse.wf/arbys.txt
- **Формат:** Text (строки с запятой: `timestamp,nodeKey`)
- **Обновление:** Каждый час на сервере browse.wf
- **Пример:** `1713607200,/Lotus/Levels/ConclaveTylacor/ConclaveTylacor_def#ReinforcedCorridors`
- **Использование:** Определяет текущие и будущие арбитражи
- **Валидация:** Парсинг с проверкой на пустые строки, корректные timestamps и nodeKeys

### 2. **Информация о регионах** (Необходимо)
- **URL:** https://browse.wf/warframe-public-export-plus/ExportRegions.json
- **Формат:** JSON
- **Содержит:**
  - Координаты локаций
  - Названия регионов и систем
  - Типы врагов (фракции)
  - Типы миссий
  - Уровни врагов (минимум/максимум)
  - Данные о Тёмных секторах (бонусы)
- **Пример ключа:** `/Lotus/Levels/ConclaveTylacor/ConclaveTylacor_def#ReinforcedCorridors`

### 3. **Русские переводы** (Необходимо)
- **URL:** https://browse.wf/warframe-public-export-plus/dict.ru.json
- **Формат:** JSON (ключ → значение)
- **Содержит:**
  - Названия локаций на русском
  - Названия систем на русском
  - Названия фракций на русском
  - Типы миссий на русском
- **Пример:** `/Lotus/Language/Factions/Grineer` → `Гринер`

### 4. **Ярусы арбитража** (Необходимо)
- **URL:** https://browse.wf/supplemental-data/arbyTiers.js
- **Формат:** JavaScript код с regex-friendly структурой
- **Содержит:** Рейтинг сложности каждого узла (S, A, B, C, D, F)
- **Пример:** `"/Lotus/Levels/ConclaveTylacor/ConclaveTylacor_def#ReinforcedCorridors": "A"`

## 🔍 Проверка источников

Запустите скрипт верификации:

```bash
node lib/verify-sources.js
```

Этот скрипт:
- ✓ Проверит доступность всех источников
- ✓ Выведет статус каждого источника
- ✓ Покажет размер загруженных данных
- ✓ Выведет примеры данных

## 🚫 Что НЕ является хардкодом

### В constants.js:
- ✓ **Маппинги эмодзи** - это UI элементы, не данные
- ✓ **Конфиги цветов** - это визуальное оформление
- ✓ **Уязвимости фракций** - это статичные правила игры, не текущие данные
- ✓ **Переводы типов миссий** - это наименования, не динамические данные

### В других файлах:
- ✓ Никаких примеров арбитражей
- ✓ Никаких тестовых данных
- ✓ Никаких mock данных
- ✓ Только функции для обработки данных

## 📈 Процесс загрузки данных

```
index.js:
  ↓
  await loadData() (каждые 60 секунд)
  ↓
  Promise.all([
    axios.get(ExportRegions.json),
    axios.get(dict.ru.json),
    axios.get(arbyTiers.js),
    axios.get(arbys.txt)
  ])
  ↓
  Кэширование в памяти
  ↓
  Функции получают из кэша
```

## ✅ Готовность к использованию

- ✓ Все данные внешние
- ✓ Полная валидация парсинга
- ✓ Обработка ошибок при загрузке
- ✓ Логирование некорректных данных
- ✓ Fallback на старые данные при ошибке
- ✓ Проверка целостности данных

## 🔐 Конфиденциальность

- Никакие пользовательские данные не отправляются на browse.wf
- Бот использует стандартный User-Agent для запросов
- Все данные - это публичная информация о игре Warframe
